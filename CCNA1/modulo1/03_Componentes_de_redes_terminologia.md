## Resumen claro

El glosario de este módulo reúne los términos fundamentales que se usan en redes. Su función no es profundizar todavía, sino dar una base técnica clara para poder leer y entender correctamente los próximos contenidos.

## Definiciones principales

### DoS
Ataque de denegación de servicio que busca interrumpir o bloquear el acceso a sistemas o servicios sobrecargándolos con tráfico malicioso.

### Capa de red
Capa 3 del modelo OSI. Se encarga de permitir la comunicación entre dispositivos finales a través de redes y de dirigir los datos hacia su destino.

### Checksum
Valor numérico usado para verificar la integridad de los datos durante la transmisión.

### Default Gateway
Dirección IP del router que usa un host para enviar tráfico fuera de su red local.

### Enrutamiento
Proceso de elegir la mejor ruta para que los datos viajen de un origen a un destino.

### DHCP
Protocolo que asigna automáticamente dirección IP, máscara de subred y puerta de enlace a un host.

### DNS
Sistema que traduce nombres de dominio en direcciones IP.

### Firewall
Dispositivo o software que filtra tráfico entrante y saliente según reglas de seguridad.

### FTP
Protocolo de transferencia de archivos entre hosts a través de una red basada en TCP.

### HTTPS
Versión segura de HTTP que utiliza cifrado.

### Dirección IP
Número único que identifica un dispositivo en una red.

### IPv6
Versión más reciente del protocolo IP, creada para resolver la limitación de direcciones de IPv4.

### ISP
Proveedor de servicios de Internet.

### Kernel
Parte central del sistema operativo que gestiona recursos de hardware y software.

### LAN
Red de área local, limitada a una zona pequeña como una casa u oficina.

### Dirección MAC
Identificador físico único de una interfaz de red.

### Malware
Software malicioso diseñado para dañar, robar información o comprometer sistemas.

### Modelo OSI
Modelo conceptual de 7 capas que explica cómo funciona la comunicación en red.

### Modelo TCP/IP
Modelo de 4 capas utilizado para comprender las comunicaciones en Internet.

### Peer-to-Peer
Modelo de intercambio directo entre dispositivos sin intermediario central.

### Packet
Bloque de datos transmitido a través de una red.

### Ping
Herramienta de diagnóstico que permite verificar conectividad.

### Protocolo
Conjunto de reglas que permite la comunicación entre dispositivos.

### Red de computadoras
Conjunto de dispositivos interconectados que se comunican usando estándares comunes.

### Router
Dispositivo que conecta redes distintas y decide la ruta del tráfico.

### Shell
Interfaz que permite interactuar con el sistema operativo.

### CLI
Interfaz de línea de comandos basada en texto.

### Subnet Mask
Valor que separa la parte de red y la parte de host en una dirección IP.

### Subnetting
División de una red IP en subredes más pequeñas.

### Switch
Dispositivo de red que reenvía tráfico dentro de una LAN utilizando direcciones MAC.

### TCP/IP
Conjunto de protocolos usado en Internet y en muchas redes locales.

### Topologías de red
Formas físicas o lógicas en las que se conectan los dispositivos en una red.

### WAN
Red de área amplia que conecta redes a gran distancia.

## Ejemplo práctico

Cuando una computadora entra a una página web:
- usa DNS para traducir el nombre del sitio
- usa su default gateway para salir de la red local
- el router enruta los paquetes
- HTTPS protege la comunicación
- los paquetes llegan al servidor por la red
