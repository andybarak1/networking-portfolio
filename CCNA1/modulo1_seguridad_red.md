# CCNA1 – Módulo 1
Fundamentos de seguridad de red

## Objetivo
Comprender los fundamentos de seguridad de red, amenazas y vulnerabilidades.

## Conceptos clave
- Seguridad de red
- Amenazas
- Vulnerabilidades
- Protección contra ataques
- Verificación de conectividad
## Ejemplo práctico

Un administrador de red configura un firewall para bloquear accesos no autorizados a un servidor web.

Si no se implementa seguridad:
- un atacante puede escanear puertos abiertos
- identificar servicios vulnerables
- explotar el sistema

## Herramientas relacionadas

- Nmap (escaneo de red)
- Wireshark (análisis de tráfico)
- Firewall (protección)

## Escenario real

Un atacante realiza un escaneo de red y detecta que el puerto 22 (SSH) está abierto.

Si el sistema no tiene protección:
- puede intentar fuerza bruta
- obtener acceso al sistema
